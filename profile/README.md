# PieTeams

**Enterprise AI software for teams.**

We build lightweight, local-first AI tools for knowledge management, productivity, and agent infrastructure.

## Projects

| Project | Description |
|---|---|
| [**PieKBS**](https://github.com/pieteams/piekbs) | A local-first knowledge base engine for AI agents — distills raw documents into structured Markdown wiki, searchable via MCP |
| [**PieMDM**](https://github.com/pieteams/piemdm) | Enterprise Master Data Management system — data governance, modeling, and integration built with Go + Vue.js |

## Philosophy

- **Local-first** — your data stays on your infrastructure
- **Agent-native** — designed for AI agents, not just humans
- **Single binary** — no external dependencies, easy to deploy
- **Open source** — MIT licensed
